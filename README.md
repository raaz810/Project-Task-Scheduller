📌 Task Scheduler Web App

A simple and efficient Task Scheduler Web Application that helps users manage daily tasks, track progress, and stay organized.

🚀 Live Demo

🔗 Website: https://raaz810.github.io/Project-Task-Scheduller/

📖 Overview

The Task Scheduler is designed to help users:

Add and manage tasks easily
Track task completion status
Organize daily workflow efficiently

This project demonstrates practical implementation of frontend development concepts and user-friendly UI design.

✨ Features
✅ Add new tasks
📝 Edit existing tasks
❌ Delete tasks
✔️ Mark tasks as completed
📋 Clean and responsive UI
⚡ Fast and lightweight
